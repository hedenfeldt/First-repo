Div tags

  • Div tags funktion är att struckturera upp html kod .
  • struckturera upp element som vi tycker hör ihop
  • gruppera som content innanför Div taggen
  • användabart när man använder CSS
  • för att sätta styling på en div skriver du style="background: Color;"efter div taggen men innan för '>'

P tags är en Paragraf tag, vill du ha utfylnad i denna kan du skriva ''lorem'' för dvs random ord

UL Tags

  • UL = UnordedList
  • med UL tags gör du en punktad lista
  • med Li tags skriver du innehållet i listan
  • Li används innanför ul taggen
  • höger klicka på webläsern och inspektera

OL Tags

  1. OL = OrdedList
  2. du kan använda OL tags för numrerad lista
  3. med Li tags skriver du innehållet i listan
  4. Li används innanför ol taggen
  5. höger klicka på webläsern och inspektera

Classes and ID

  1. Classes & ID använd innanför tags

  6. '.' används för classes i stylesheet
